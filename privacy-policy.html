Chrome Extension Privacy Policy
Last updated: May 2026

🔒 Bili Clipper 不收集、不上传、不分享任何用户数据。所有数据仅存储在您本地设备上。

我们收集哪些数据

Bili Clipper 仅在您的浏览器本地存储以下信息：

  • 用户设置：Obsidian Vault 路径、笔记模板等偏好设置
  • Clip 历史：您手动 Clip 的 B 站视频记录（仅保存在本地）

以上数据通过浏览器原生的 chrome.storage.local API 存储，仅限您本人访问，不会离开您的设备。

我们不做什么

  • 不收集任何个人身份信息（姓名、邮箱、账号等）
  • 不追踪您的浏览行为或观看记录
  • 不向任何第三方服务器上传数据
  • 不使用任何分析或广告 SDK
  • 不在扩展卸载后保留任何数据

本地服务器

Bili Clipper 需要在您的 Mac 上运行一个本地 Python 服务（端口 27182），用于将字幕写入 Obsidian。此服务：

  • 仅监听 localhost，无法被外部网络访问
  • 不发送任何数据到互联网
  • 由您手动安装和控制，可随时卸载

数据存储与删除

如需删除所有本地数据，可在 Chrome 扩展设置中点击"清除数据"，或直接卸载扩展。卸载后，chrome.storage.local 中的数据会被浏览器自动清除。

联系我们

如有任何隐私相关问题，欢迎通过 GitHub Issues 联系我们。

🔒 Bili Clipper does not collect, upload, or share any user data. All data stays on your local device only.

What We Store

Bili Clipper stores only the following information locally in your browser:

  • User settings: your Obsidian Vault path, note templates, and other preferences
  • Clip history: a local log of videos you have manually clipped

All data is stored via the browser-native chrome.storage.local API, accessible only to you, and never leaves your device.

What We Do Not Do

  • We do not collect any personally identifiable information (name, email, account, etc.)
  • We do not track your browsing behavior or watch history
  • We do not upload data to any remote server
  • We do not use analytics or advertising SDKs
  • We do not retain any data after the extension is uninstalled

Local Server

Bili Clipper requires a lightweight local Python server (port 27182) on your Mac to write subtitles into Obsidian. This server:

  • Listens on localhost only — not reachable from the internet
  • Does not send any data to the internet
  • Is installed and controlled by you, and can be uninstalled at any time

Data Deletion

To delete all local data, click "Clear Data" in the extension settings, or simply uninstall the extension. Chrome automatically clears chrome.storage.local data upon uninstall.

Contact

For any privacy-related questions, please open an issue on GitHub Issues.

Bili Clipper is an open-source Chrome extension. This policy applies to the extension only and does not cover Bilibili (bilibili.com) or any third-party services.
